Project 2 — Post-Earthquake Assistant

HCI Chatbot Speech I/O

This project is about designing an agent that can facilitate disaster management. What we create is an AI-boost chatbot that can actually help people after the disaster.

My Contribution
  • design and write all the source code(690 lines of domain-specified AI chatbot) of the prototype--AI Chatbot; based on the designed ideas provided by my groupmate.
  • Test all functions of the Chatbot and modify it to make it more fit to design and more user-comfortable.
    • Functions includes: audio inputs and audio outputs, resources location, empathetic emotion,help guide for new users, volunteer guide, instructions for injury, instruction for depression emotion handling..etc.
  • Usability Testincluding: design the test tasks, invite people, monitor the test, collect results, analyze results, write reports.
▶ View Demo of all functions
reflection
  • Designing and coding the AI chatbot let me learn a lot about AI API protocol--how to connect it, how to actually implement it in our own project
  • Clearly understand what the whole design process is and how it flows, as well as how to cooperate with teammates in the design process.
  • When encoutering problems within group members, communication as early as possible is the most useful way.

Project 3 — Example Project Title

HCI Unity XR

This project focuses on designing and evaluating a system for elderly people who live alone to make them stay interactive with the outside world using XR method.

My Contribution
  • Ideation part for the project, brainstormed with my groupmate and came up with the ideas for the prototype.
  • Developed and tried Unity XR to build the solution(build the scene,character,objects,interactive buttons,etc.).
  • Design the virtual scene, character, objects and build the interactive buttons and their functions.
  • Design the interaction flow for the user.
  • test and analyze my Unity XR solution to see if it actually suitable for our target users.
  • Design and orgnize the presentation and structure of the ideation part of the project.
▶ View Demo / PDF / Prototype
reflection
  • Learned how to build the XR project in Unity. In the process of building the project, I searched and learned a lot about basic knowledge of Unity.
  • Brainstorming with my groupmate helped us generate more creative and practical ideas. I also realised the value of evaluating ideas based on user needs early in the process.
  • Testing the XR solution showed me that some features that felt intuitive to me were not suitable for target users.
